#### A comprehensive list of my favourite Atom packages from 2015



### [ask-stack](https://github.com/Chris911/Ask-Stack-Atom.git)

Quickly get answers and code samples from Stack Overflow in Atom

### [atom-jade](https://github.com/merrihew/atom-jade)

Jade language support in Atom

### [linter](https://github.com/atom-community/linter.git)

Linter is a base linter provider for the Hackable Atom Editor.

It provides a top-level API to its consumer that allows them to visualize errors and other types of messages, easily.
![linter](https://camo.githubusercontent.com/70b6e697c9d793642414b4ea6d08dbb9678877b3/687474703a2f2f672e7265636f726469742e636f2f313352666d6972507a322e676966)

### [linter-coffee-variables](https://github.com/markwainwright/atom-linter-coffee-variables.git)

This is a plugin for Linter that does some things that CoffeeLint can't: detecting variables that are used without being defined, and defined variables that aren't used.
It's best used in conjunction with linter-coffeelint.

### [linter-coffeescript](https://github.com/hokaccha/linter-coffeescript.git)

Lint CoffeeScript on the fly, using coffee

### [linter-eslint](https://github.com/AtomLinter/linter-eslint.git)

Lint JavaScript on the fly, using ESLint

### [markdown-preview-plus](https://github.com/Galadirith/markdown-preview-plus.git)

Better markdown preview in atom (optional pandoc support)
![markdown-preview-plus](https://raw.githubusercontent.com/Galadirith/markdown-preview-plus/master/imgs/mpp-full-res-invert.png)

A minimap binding for the highlight-selected package
![minimap-highlight-selected](https://github.com/atom-minimap/minimap-highlight-selected/raw/master/screenshot.png)

### [minimap-pigments](https://github.com/abe33/minimap-pigments.git)

An Atom plugin to display pigments colors in the Minimap.
![minimaps-pigments](https://github.com/abe33/minimap-pigments/raw/master/screenshot.png)

### [pigments](https://github.com/abe33/atom-pigments.git)

A package to display colors in project and files.
![pigments](https://github.com/abe33/atom-pigments/raw/master/resources/pigments.gif)

### [preview](https://github.com/Glavin001/atom-preview.git)

Ultimate previewer of source code in Atom.

### [react-snippets](https://github.com/webbushka/atom-react-snippets.git)

An Atom snippet library for React, React Router and Flux. This library uses ES6 syntax, if you would prefer ES5 or Coffeescript snippets please use the Atom React Package.

### [set-syntax](https://github.com/lee-dohm/set-syntax.git)

Creates easy Command Palette commands for setting the syntax of the current file
![set-syntax](https://raw.githubusercontent.com/lee-dohm/set-syntax/master/set-syntax.gif)

### [stacktrace](https://github.com/smashwilson/stacktrace.git)

Navigate stacktraces within Atom.

This is one of my favourite packages in the list. It is very useful when debugging.
![stacktrace](https://cloud.githubusercontent.com/assets/17565/4100060/aa761e90-307e-11e4-83c8-e4bf04c20d95.gif)

### [synced-sidebar](https://github.com/peterdotjs/atom-synced-sidebar.git)

Based on Sublime Text SyncedSideBar plugin. Active tab & sidebar are synced - sidebar entry automatically scrolls into view.
![synced-sidebar](https://raw.githubusercontent.com/peterdotjs/atom-synced-sidebar/master/synced-sidebar-min.gif)

### [toggle-quotes](https://github.com/atom/toggle-quotes.git)

Quickly toggle between single and double quotes
